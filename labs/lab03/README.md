malloc выделяет блок памяти заданного размера в байтах, не инициализируя его

malloc выделяет память без инициализации, а calloc выделяет и обнуляет её, плюс принимает количество элементов и их размер.

realloc изменяет размер уже выделенного блока памяти (увеличивает или уменьшает), при необходимости копируя данные в новое место.

При утечке памяти программа теряет ссылку на выделенную область, и память не освобождается до завершения программы, что может привести к её нехватке.

free нужен, чтобы вернуть выделенную динамически память обратно операционной системе и избежать утечек.

При двойном освобождении памяти программа может аварийно завершиться, получить повреждение кучи или непредсказуемое поведение.

Проверить, что malloc вернул NULL, можно сравнением указателя с NULL: if (ptr == NULL) { printf("Ошибка: malloc вернул NULL\n"); }.

Статический массив имеет фиксированный размер, известный на этапе компиляции, а динамический выделяется во время выполнения через malloc/calloc и может изменять размер (через realloc).
